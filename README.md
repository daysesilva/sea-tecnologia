Sea Tecnologia

Requisitos para rodar a aplicação.

    VirtualBox
    Vagrant
    Puttygen (Windows)
    Putty (Windows)

Criado a máquina virtual rodar os seguitens códigos

1 - Abre console e digite:
    
    git clone https://github.com/daysesilva/sea-tecnologia.git
    cd sea-tecnologia
    vagrant up

2 - Conecte via ssh:

    python app.py

3 - Acesse http://localhost:5000/
